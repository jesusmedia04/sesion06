BOX MODEL O MODELO DE CAJA EN CSS

Que es Box Model

Describe como se estructuran y dimensionan los elementos en una pagina mediante cajas rectangulares

Que es una caja en box model

Cada elemento es una caja, y cada caja esta compuesta por cuatro areas principales:
content, padding, border y margin.

Elementos y/o colores y/o capas de una caja box model

Content - azul

La parte interna de la caja que contiene el contenido real del elemento, como texto, imágenes u otros elementos HTML

Content

Padding - verde

El espacio entre el contenido y el borde. Se utiliza para aumentar el espacio interno de la caja.

Padding

Border - amarillo

La línea que rodea el contenido y el relleno. Puede tener un ancho, estilo y color específicos.

Border

Margin - naranja

El espacio entre la caja y otros elementos adyacentes en la página. Se utiliza para aumentar el espacio externo de la caja.

Margin

Que son las propiedades width y height

width

Modifica el ancho del elemento, en este caso 200 px de ancho

height

Modifica el alto del elemento, en este caso 50 px de alto

Cual es la importancia de usar box-sizing: border-box ; y con que selector se utiliza

Es importante porque incluye tanto el relleno como el borde en el tamaño total del elemento,
lo que facilita el control del diseño y evita sorpresas en el diseño

Definicion y ejemplo de Padding

Se refiere al espacio entre el contenido de un elemento y su borde.

Padding

Definicion y ejemplo de Border

Se refiere a la línea que rodea el contenido y el relleno de un elemento.

Border

Definicion y ejemplo de Margin

Se refiere al espacio entre el borde de un elemento y los elementos circundantes.

Margin

Que es el colapso de margenes verticales

Ocurre cuando los márgenes verticales de dos elementos adyacentes se combinan en lugar de acumularse.

En este caso, en lugar de que la distancia entre div1 y div2 sea la suma de los márgenes (20px + 30px = 50px),
el colapso de márgenes tomará el máximo de los dos márgenes, que en este ejemplo sería 30px. Entonces, la distancia final entre los elementos será de 30px.

Que es el margen negativo

un margen negativo significa que el elemento se desplazará hacia fuera de su contenedor
o se superpondrá a otros elementos. Normalmente, el margen se utiliza para agregar espacio alrededor de un elemento, pero cuando se establece como un valor negativo, se invierte este comportamiento.
